# Content Standards for Newcomer Living / 加拿大生活工具箱

This document defines the quality bar for future guides, tools, and category pages. The site should help Chinese-speaking newcomers, students, workers, and families make practical Canada life decisions without becoming a set of thin, generic AI-like articles.

## 1. Core editorial principles

Every page should be:

- **Practical:** solves a real newcomer problem with concrete next steps.
- **Chinese-first:** public-facing content should be natural, clear Chinese unless a specific English term is useful.
- **Canada-specific:** include Canadian context such as provinces, cities, schools, official processes, weather, housing, transit, banking, or documentation.
- **Cautious:** avoid strong legal, tax, immigration, medical, insurance, or financial claims.
- **Source-aware:** link to official or credible sources when discussing rules, government processes, school registration, driving, housing rights, tax, customs, insurance, or health coverage.
- **Useful beyond search traffic:** a newcomer should be able to bookmark the page and use it as a checklist or decision aid.

## 2. Minimum guide structure

A serious guide should generally include:

1. Clear target reader
2. Quick answer or summary box
3. Specific Canadian context
4. Practical step-by-step explanation
5. One or more realistic examples or scenarios
6. Checklist, table, comparison, or decision guide
7. Common mistakes or risk reminders
8. Related internal links
9. Related tool links where useful
10. Official or credible source links when rules/processes are discussed
11. Updated date and reading time
12. Page-level disclaimer box

Shorter guides can be concise, but they should still answer a focused user question and suggest a useful next step.

## 3. Pre-publication article checklist

Before publishing or expanding a guide, confirm:

- [ ] Would a Chinese newcomer in Canada bookmark this page?
- [ ] Does the page answer one clear question?
- [ ] Does it add value beyond a generic AI answer?
- [ ] Does it include Canadian context, not just universal advice?
- [ ] Does it include concrete examples, ranges, tables, or checklists where helpful?
- [ ] Does it explain assumptions and limitations?
- [ ] Does it avoid unsupported legal, tax, immigration, insurance, medical, or financial claims?
- [ ] Does it use cautious wording such as “通常情况下”, “以下为一般参考”, “建议进一步确认”, and “以官方信息为准”?
- [ ] Does it link to official or credible sources when rules/processes are discussed?
- [ ] Does it link to the next useful guide, tool, or category page?
- [ ] Does it include a relevant disclaimer box?
- [ ] Does it avoid development/prototype language such as “MVP”, “placeholder”, “coming soon”, or “draft” in public copy?

## 4. Sample guide outline

Use this outline for a high-value guide such as `加拿大租房押金和首月房租怎么理解？`:

```md
# 加拿大租房押金和首月房租怎么理解？

更新日期：YYYY-MM-DD · 阅读时间：X 分钟

## 快速摘要
- 以下为一般参考，不同省份规则可能不同。
- 新人租房要区分首月房租、末月房租、押金、钥匙押金和其他费用。
- 签约和付款前应查看当地官方租房信息，并保留书面记录。

## 这篇指南适合谁？
说明目标读者：刚到加拿大、没有本地租房经验、准备看房或签约的人。

## 常见付款项目说明
用表格解释项目、常见含义、需要确认的问题。

## 真实场景示例
例如：单人留学生合租、三口之家整租、公寓物业申请。

## 签约前检查步骤
1. 确认房东或物业身份
2. 确认费用是否可退
3. 确认租期和付款方式
4. 保存租约、收据和沟通记录

## 常见误区
- 只看月租，不看总入住成本
- 没有看当地规则就支付大额押金
- 相信口头承诺，没有写入租约或邮件

## 官方或参考来源
- Provincial tenancy authority
- City or legal aid tenant information page

## 下一步
- 加拿大租房看房清单
- 加拿大租房合同需要注意什么？
- 加拿大月生活成本计算器

## 免责声明
本页面仅供一般信息参考，不构成法律、移民、税务、财务、保险或其他专业建议。
```

## 5. Writing style rules

- Write in a practical, calm, newcomer-friendly tone.
- Prefer short paragraphs and clear section headings.
- Use examples such as “单人留学生”, “三口之家”, “刚到安省”, “Thunder Bay”, or “Toronto/GTA” when relevant.
- Use tables for comparisons, budgets, checklists, and decision criteria.
- Avoid exaggeration, fear-based language, or promises such as “一定可以”, “保证”, “最便宜”, or “官方认可”.
- Avoid thin intros such as “加拿大是一个美丽的国家” unless directly useful.
- Avoid keyword stuffing. Keywords should fit naturally into useful content.
- Keep public copy production-ready and trustworthy.

## 6. Source citation rules

Use official sources where possible for:

- immigration or status-related processes
- SIN, tax, CRA, Service Canada, IRCC, CBSA topics
- school registration and education board requirements
- tenancy rules, deposits, leases, and landlord/tenant rights
- driver licensing, winter driving, vehicle registration, insurance
- health coverage or medical access
- customs, travel, food, medicine, cash declaration, or import rules

Source display should be simple:

```md
## 官方或参考来源
- Service Canada — Social Insurance Number
- Ontario.ca — Renting in Ontario
- CBSA — Travelling to Canada
```

Rules:

- Prefer official government, school, municipality, transit, university, bank, or insurance regulator sources.
- If using non-official sources, explain why they are credible.
- Do not copy official text wholesale; summarize and link.
- When rules vary by province/city/status, say so clearly.
- Add/update the page `updated` date after material source review.

## 7. Internal linking rules

Every major guide should include useful next steps:

- Link to the parent category.
- Link to 2–4 related guides.
- Link to a related tool when it helps the reader act.
- Link from cluster articles to each other when the user journey is connected.

Examples:

- Cost guide → monthly cost calculator, newcomer checklist, renting guide.
- Renting guide → deposit guide, viewing checklist, lease guide.
- Student guide → SIN guide, part-time resume guide, phone plan guide.

Internal links should be useful, not random.

## 8. Disclaimer usage rules

Use page-level disclaimers on:

- all guide pages
- calculators and checklists
- government document topics
- immigration, tax, school, health, insurance, driving, customs, travel, and housing-rights topics

Preferred wording should remind users that:

- the site is not a government website
- information is general only
- it is not legal, immigration, tax, financial, medical, insurance, or other professional advice
- rules can vary by province, city, school, institution, or personal status
- important decisions should be verified with official sources or professionals

## 9. Tool and calculator content standards

Interactive tools should include:

- clear input labels
- explanation of assumptions
- cautious result wording
- links to explanatory guides
- links to the main disclaimer or a page-level disclaimer
- no collection of unnecessary personal information

For budget calculators, always state that outputs are estimates only.

## 10. Category page standards

Category pages should not be only lists. A useful category page should include:

- clear category introduction
- “Start here” reading path
- featured guides
- related tools
- related categories
- FAQ or common questions where useful
- disclaimer if the topic is sensitive

## 11. Quality bar before adding new pages

Do not publish a new guide if it only contains generic paragraphs. Either improve it to meet this standard or keep it out of public navigation until it has practical value.

A page is ready when it helps a real reader answer: **What should I do next, what should I check, and where can I verify important information?**
